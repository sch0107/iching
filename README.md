# 易经起卦 · 中国传统占卜

基于 React + Vite 构建的中国传统占卜网页应用，支持七种占卜方式。

**[在线体验 →](https://sch0107.github.io/iching/)**

---

## 功能概览

### 七种占卜模式

| 模式 | 简介 |
|------|------|
| **易经** | 完整六十四卦起卦，含变爻与变卦 |
| **梅花易术** | 邵雍时间起卦法，起本卦、动爻、变卦 |
| **铜钱** | 单枚铜钱投掷，快速问是非，支持单掷/多掷/连续掷三种模式 |
| **圣筊** | 两块月牙形木筊掷筊问卜，圣筊/笑筊/阴筊三种结果，支持单掷/多掷/连续掷三种模式 |
| **小六壬** | 依当前时间推算六神课 |
| **大六壬** | 生辰八字起课，输入出生年月日时 |
| **塔罗牌** | 78 张牌，单张、三张、五张、凯尔特十字（十张）牌阵，含逆位 |

### 多语言支持

- 简体中文
- 繁體中文
- English

### 其他功能

- 时区选择（小六壬 & 梅花易术）- UTC+8 或本地时区
- 一键复制占卜结果，可直接粘贴给 AI 解读
- 导出 `.txt` 文件（易经模式）
- 适配移动端布局
- "随机"模式快速体验
