# 易经起卦 · 中国传统占卜

基于 React + Vite 构建的中国传统占卜网页应用，支持五种占卜方式。托管于 GitHub Pages。

**[在线体验 →](https://sch0107.github.io/iching/)**

---

## 功能概览

### 六种占卜模式

| 模式 | 简介 |
|------|------|
| **易经** | 完整六十四卦起卦，含变爻与变卦 |
| **铜钱** | 单枚铜钱投掷，快速问是非 |
| **小六壬** | 依当前时间推算六神课 |
| **梅花易术** | 邵雍时间起卦法，起本卦、动爻、变卦 |
| **大六壬** | 生辰八字起课，输入出生年月日时 |
| **塔罗牌** | 78 张牌，单张、三张、五张、凯尔特十字（十张）牌阵，含逆位 |

### 多语言支持
- 简体中文
- 繁體中文
- English

所有文字（包括六十四卦卦名、卦辞、八卦名、六壬神名、78 张塔罗牌名及牌义）均存于语言文件中，组件内无硬编码译文。

### 时区选择（小六壬 & 梅花易术）
- **UTC+8**（默认）：以北京时间计算时辰，传统方法推荐
- **本地**：使用设备本地时区

### 其他
- 一键复制占卜结果，可直接粘贴给 AI 解读
- 导出 `.txt` 文件（易经）
- 适配移动端布局

## 占卜方法简介

### 易经（周易起卦）
周易是中国最古老的占卜典籍，通过摇卦得到由六条爻组成的卦象（共六十四卦）。本应用模拟"蓍草法"概率，每爻以四营十八变确定阴阳，并区分老阴、老阳（变爻）。若有变爻，则额外生成变卦，两卦合参以判断事物的当下状态与走向。

### 铜钱（一枚铜钱）
以单枚铜钱正反面代表阴阳，简洁直接地回答是非问题。正面为阳，主吉，事可行；反面为阴，主否，宜缓行。适合需要快速决断的场合。

### 小六壬
小六壬是民间流传的简易时间占卜术，以月、日、时三数依次推算，落于大安、留连、速喜、赤口、小吉、空亡六神之一。六神各有吉凶含义，可判断当前事宜的顺逆。起课方法：以月数为起点，加日数后再加时支数，除以六取余，对应所得神课。

### 梅花易术（时间起卦）
梅花易术为宋代邵雍所创，以先天八卦数配合时间推算卦象，无需蓍草或铜钱。起卦步骤如下：

| 步骤 | 公式 |
|------|------|
| 年数 | 当前年份末两位 |
| 上卦 | （年 + 月 + 日）÷ 8 取余，0 作 8 |
| 下卦 | （年 + 月 + 日 + 时支）÷ 8 取余，0 作 8 |
| 动爻 | （年 + 月 + 日 + 时支）÷ 6 取余，0 作 6 |

先天八卦数：1=乾，2=兑，3=离，4=震，5=巽，6=坎，7=艮，8=坤。上下卦合成本卦，动爻变爻后得变卦，并据动爻位置判断体卦（不动者）与用卦（含动爻者），以体用关系论吉凶。

### 大六壬（生辰八字）
大六壬是中国古代三大占卜术之一（三式之一），需输入生辰八字（出生年月日时）。本应用实现了完整的大六壬体系，严格遵循传统推算方法，包括：

- **三傳（初传、中传、末传）**：使用传统九宗门方法推算三传
  - 贼克法：仅在只有一个下克上时使用，多个时转入涉害法
  - 涉害法：比较多个贼克的涉害深浅，取最深者（传统"沿途被克地支数"计算）
  - 遥克法：四课内部无克时，看日干与四课各支之间的克关系，优先取日克支
  - **八专法**：日干为甲或己时，取日支上神为初传（新增实现）
  - **别责法**：四课中有两个相同地支时，取该地支上神为初传（新增实现）
  - **昴星法**：昼取酉上神，夜取卯上神（新增昼夜选择）
- **四課**：按传统方法从日干寄宫推算四课，结合天盘上神，分析干支关系与五行生克
- **天盤地盤**：采用月将加时法推算天盘旋转，月将落在时支位置上，地盘固定十二地支
  - **天盤**：仅包含十二地支位置，用于四课上神计算
  - **十二天将**：独立计算，昼占顺时针排布，夜占逆时针排布（修复概念混淆）
- **五行分析**：全面分析五行生成与克制关系，判断元素平衡
- **空亡**：根据日干地支在六十甲子中的旬位计算空亡地支，准确判断落空之象
- **局（上局、中局、下局）**：根据日干月支关系确定局势
- **十二将**：完整实现十二天将（贵人、螣蛇、朱雀、六合、勾陈、青龙、天空、白虎、太常、玄武、太阴、天后）
  - 昼贵人：甲戊庚牛羊，乙己鼠猴乡，丙丁猪鸡位，壬癸蛇兔藏，六辛逢马虎
  - 夜贵人：乙己夜贵→申，丙丁夜贵→酉，壬癸夜贵→卯，辛夜贵→寅，甲戊庚夜贵同昼
- **六亲**：按五行生克关系准确计算（生我者父母，我生者子孙，克我者官鬼，我克者妻财，比和者兄弟）
- **月将（Monthly General）**：据二十四节气准确判定月将（支持2024-2026年精确日期）
  - 雨水后起亥将、春分后起戌将、谷雨后起酉将等，每年节气日期略有不同
- **寄宫（Temporary Residence）**：日干寄宫映射（甲→寅、乙→辰、丙→巳、丁→未、戊→巳、己→未、庚→申、辛→戌、壬→亥、癸→丑）
- **五虎遁**：正确计算月干，甲己之年丙作首
- **综合卦象**：基于所有因素计算整体运势（大吉、吉、平、凶、大凶）
- **日占/夜占**：根据时辰区分日间与夜间占卜，贵人及天将位置有不同

### 塔罗牌
塔罗牌源于欧洲，共 78 张：22 张大阿尔卡那（编号 0–21，涵盖愚者至世界）与 56 张小阿尔卡那（权杖、圣杯、宝剑、星币四组，各 14 张）。本应用提供单张、三张、五张、凯尔特十字（十张）多种牌阵，并可开启逆位选项。每张牌含正位与逆位关键词及释义，结果支持一键复制。

## 项目结构

```
├── App.jsx              # 根组件：导航栏、模式路由、语言切换
├── i18n.js              # i18next 配置，合并各语言文件
├── main.jsx             # React 入口
├── index.html           # Vite 模板
├── vite.config.js       # base: "/iching/"
├── modes/
│   ├── IChing.jsx       # 易经起卦
│   ├── CoinToss.jsx     # 铜钱占卜
│   ├── XiaoLiuRen.jsx  # 小六壬（含时区选择）
│   ├── MeiHua.jsx       # 梅花易术（含时区选择）
│   ├── Da6.jsx          # 大六壬（生辰八字）- 完整三傳四課天盤地盤体系
│   ├── daLiuRenData.js # 大六壬数据结构 - 十二将、五行、空亡、局等
│   └── Tarot.jsx        # 塔罗牌占卜
│   └── tarotData.js     # 78 张牌结构数据 + Fisher-Yates 洗牌
└── locales/
    ├── en.json          # 英文：界面文字 + 六十四卦 + 八卦 + 六壬神
    ├── zh-Hans.json     # 简体中文
    ├── zh-Hant.json     # 繁體中文
    ├── tarot-en.json    # 英文塔罗牌名及牌义
    ├── tarot-zh-Hans.json  # 简体中文塔罗
    └── tarot-zh-Hant.json  # 繁體中文塔羅
```
